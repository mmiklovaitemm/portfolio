# Portfolio Website — Adam Keyes Layout

A single-page portfolio website built with **React + Vite** and styled using **Tailwind CSS**.  
This project is based on the Adam Keyes portfolio design from Frontend Mentor and focuses on responsive layout, clean UI, and smooth user interactions.

## Features

- Fully responsive layout (mobile, tablet, desktop)
- Sections: Hero, Skills, Projects, Contact, Footer
- Smooth scroll navigation between sections
- “Back to top” button that appears when reaching the Contact/Footer section
- External profile links open in a new tab

## Built With

- React (Vite)
- Tailwind CSS
- JavaScript (ES6)
